# SecureVault 密码管理器

一个基于 Electron 构建的现代化、严肃风格的本地密码管理器。它具有深色模式 UI，支持本地文件加密存储（模拟），并提供了类似原生应用的窗口体验。

## 🛡️ 功能特点

- **本地优先**：数据存储在您指定的本地 JSON 文件中，不上传云端。
- **严谨 UI**：采用深色蓝灰配色，无 Emoji，专业视觉设计。
- **AES 加密结构**：虽然目前是演示版本（使用 Base64+混淆），但代码结构已预留了接入 Node.js `crypto` 模块进行 AES-256 加密的位置。
- **跨平台**：支持 Windows、macOS 和 Linux。
- **窗口管理**：自定义的顶部标题栏和窗口控制按钮。

## 📦 目录结构

- `main.js`: **主进程**。负责创建窗口、处理系统事件以及读写本地文件。
- `renderer.js`: **渲染进程**。负责 UI 交互、表格渲染和用户操作逻辑。
- `preload.js`: **预加载脚本**。安全地将主进程的文件读写能力暴露给渲染进程。
- `index.html`: 界面布局与样式。
- `package.json`: 项目配置与依赖管理。

## 🚀 快速开始

### 1. 环境准备

确保您的电脑上已安装 [Node.js](https://nodejs.org/) (建议版本 16+)。

### 2. 安装依赖

在项目根目录打开终端（命令行），运行：

```
npm install
```

### 3. 开发模式运行

这会启动应用并打开开发者工具窗口：

```
npm start
```

### 4. 打包构建

生成可执行文件（.exe / .dmg）：

```
npm run dist
```

## ⚠️ 安全提示

当前版本的加密逻辑（`mockEncrypt`）仅用于演示 UI 效果。在用于生产环境存储真实敏感密码之前，请务必在 `main.js` 中替换为真实的 `crypto.createCipheriv` 实现。
